报错 #

如果你遇到了报错，请阅读以下内容或在搜索框内搜索关键字

 

启动游戏时 #

HMCL提示下载文件失败 #

请点击设置切换下载源为mcbbs

游戏崩溃 #

  • 请调大游戏运存至3000MB（最少）
  • 请不要私自添加mod，由此引发的mod冲突导致的崩溃我们概不负责
  • 请使用正确的java，不要从某某下载站下载，推荐去官网 (opens new window)下载正版java

游戏中 #

无法解析主机名 #

请使用admin.5imczy.cn:60006作为服务器地址

掉线并提示Timed out（或连接超时） #

可能是网络波动导致，如非大部分玩家集体出现此报错请尝试刷新DNS、重启路由等操作

out of memory #

您为游戏分配的运行内存不足，请调大运存

无法进入服务器，服务器状态显示为“V 旧版” #

服务器正在重启中，请稍后

提示 Server Mod rejections: xxxx #

您的客户端缺少某些必要mod，请不要私自删减mod

游戏崩溃 #

原因可能为：

  • 触发模组bug
  • 过于卡顿导致未响应造成崩溃
  • 卡区块
  • 等等

如出现非卡顿导致的游戏崩溃请反馈给腐竹或管理人员，并提供详细信息
