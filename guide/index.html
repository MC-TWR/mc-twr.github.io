使用 #

 

下载 #

  • 请在Q群 (opens new window)内下载最新版服务器整合包压缩文件
  • 请勿随意使用他人发送的整合包文件，出问题概不负责
  • 请仔细辨别最新版整合包文件（分不清可以在群内询问）
注意，缺少java将无法运行游戏，如没有java请在群文件内或[java官网](https://java.com/)下载安装（服务器运营版本为1.12.2，对应java8）

解压 #

  • 推荐使用Bandizip (opens new window)进行解压（别问，问就好看）
  • 请务必完全解压整合包文件
  • 目录下至少应有.minecraft文件夹和启动器.exe

运行 #

双击运行启动器程序，添加账号，使用皮肤站账号登录

注意：账号为你的注册邮箱，并非你的昵称

点击开始游戏

如果你的游戏正常启动且顺利进入了主界面，恭喜你可以开始游玩
