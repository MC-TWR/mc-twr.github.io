Q&A #

 

在您提问前请先学会《提问的智慧》 (opens new window)

这个粉红色的UI是怎么回事，我能不能去掉它 #

这个界面效果是由模组PasterDream所自带的

如果您想去除他，可以使用上文提到过的Bandizip打开该模组的.jar文件

找到assets/minecraft/textures目录，删掉textures文件夹下所有内容并重启游戏即可

OpenFM/收音机怎么使用 #

您可以百度寻找音乐直链提取，从中获取音乐直链

如无法播放请尝试拆掉收音机重新放置

音乐直链通常为.mp3结尾

图片展示框/在线图片加载如何使用 #

您可以在网上寻找任意图床，上传图片后复制图片链接粘贴到展示框界面即可

服务器考不考虑加入xxx模组 #

合理的建议我们会接受的，请将想法反馈给腐竹

我能不能自己加入光影/材质包 #

可以的，但我们不推荐您使用矿物透视类材质及任何带有作弊性质的材质和光影

请随意添加作弊材质（甚至是作弊mod），请放心，管理组不会封禁您

服务器好卡啊该怎么办 #

过于卡顿可在服务器内使用/vote投票重启服务器，人数过半即可重启，不过事先要搞清楚是服务器卡还是客户端卡

QQ群内的管理都是OP么 #

不是的，有部分是摸鱼（

我拔刀剑丢了能不能补给我 #

如果是个人原因导致的拔刀剑丢失（包括但不限于丢入岩浆、强化失败等）服务器概不负责不一定负责

如果是因服务器卡顿导致回档丢失物品可以找管理人员补发

我遇到bug了该怎么办 #

遇到bug请及时向管理人员反馈，并提供详细信息，如果导致游戏崩溃请提供崩溃日志
